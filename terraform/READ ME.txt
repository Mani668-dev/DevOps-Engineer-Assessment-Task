# Terraform AWS Setup (VPC + EC2)

This Terraform script sets up a basic AWS infrastructure:

- VPC with a public subnet
- Internet Gateway and Route Table
- Security Group for HTTP and SSH
- EC2 instance (Amazon linux2 ) with public IP
- SSH Key Pair for access

---

## Prerequisites

- Terraform installed
- AWS CLI configured (`aws configure`)
- SSH key pair at `~/.ssh/id_rsa` and `~/.ssh/id_rsa.pub`

---

## How to Use

1. **Initialize**
  
---- terraform init
   
----terraform apply
To Destroy
------terraform destroy

   
   Connect to EC2
 -----ssh -i ~/.ssh/id_rsa ec2-user@<EC2_PUBLIC_IP>
